# Assets runtime validés

Ce dossier contient uniquement les fichiers réellement chargés par le jeu.

Un asset peut entrer ici seulement lorsqu’il :

- respecte `RIGHTBOUND_STYLE_V1` ;
- respecte son slot de `geometry-contract.json` ;
- obtient au moins 9/10 sur la grille d’acceptation ;
- passe les contrôles de dimensions, ratio, alpha, marges, poids et lisibilité mobile ;
- ne présente aucune frange magenta ;
- a été validé à sa taille réelle sur téléphone.

Les variantes rejetées, sources haute résolution et fichiers exploratoires sont interdits dans ce dossier.
