# Sources haute résolution

Ce dossier conserve les versions maîtres des assets approuvés ou encore en validation.

Règles :

- un sous-dossier par identifiant d’asset ;
- conserver le fichier source le moins compressé ;
- ne jamais renommer une variante rejetée comme fichier final ;
- indiquer le Style ID `RIGHTBOUND_STYLE_V1` dans le ticket associé ;
- respecter le slot défini par `geometry-contract.json`.

Les fichiers de ce dossier ne sont pas chargés directement par le jeu.
