# Assets du menu complet

Dépose directement dans ce dossier tous les sprites et éléments d’interface du menu principal.

Chemin cible : `assets/menu_full/`

Conserver exactement les noms de fichiers validés afin que les liaisons dans le code puissent être faites sans ambiguïté.

Ce fichier peut rester dans le dossier ; il sert uniquement à maintenir le dossier présent dans GitHub avant l’ajout des images.
